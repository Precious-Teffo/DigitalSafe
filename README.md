DigitalSafe is a web-based cybersecurity education platform designed to help everyday users, families, and small businesses stay safe online. It combines structured lessons, interactive simulations, and security tools into one simple, user-friendly interface.


##  Overview

DigitalSafe provides practical cybersecurity training through lessons, real-world attack simulations, and interactive security tools. The platform focuses on clear, non-technical guidance to help users recognize threats and protect their digital lives.
